# UI Patterns

## Form Patterns
- **Validation States** - Error/Success/Warning Styling
- **Progressive Disclosure** - Schritt-für-Schritt Forms
- **Auto-Save Indicators** - "Saving..." / "Saved" States

## Data Display Patterns
- **Loading States** - Skeleton Screens, Spinners
- **Empty States** - "No data yet" Illustrations
- **Error States** - "Something went wrong" Handling

## Navigation Patterns
- **Breadcrumbs** - Wo bin ich?
- **Tabs vs. Accordion** - Wann was verwenden?
- **Mobile Menu** - Hamburger vs. Bottom Tab

## Feedback Patterns
- **Toast Notifications** - Success/Error Messages
- **Confirmation Dialogs** - "Are you sure?" Modals
- **Progress Indicators** - Multi-step Processes

## Layout Patterns
- **Card Layouts** - Content-Gruppierung
- **Grid Systems** - Responsive Layouts
- **Sticky Elements** - Navigation, CTAs

## Projektspezifische Patterns

### Immobilien-spezifische Patterns
- **Parameter Input** - Eingabefelder für Immobiliendaten
- **Normalverteilung Visualisierung** - Chart-Darstellung
- **Benchmark Cards** - Vergleichswerte anzeigen
- **Export Preview** - PDF-Vorschau vor Download
